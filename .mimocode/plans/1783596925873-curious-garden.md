# Fix: Reset Angle не сбрасывает угол

## Root Cause

При правом клике на FlowTextBlkItem Qt **не выделяет** элемент (выделение только по левому клику). Контекстное меню (включая "Reset Angle") показывается, но `onResetAngle()` вызывает `selected_text_items()` → `self.selectedItems()` → пустой список → действие не выполняется.

Эта же проблема затрагивает **4 действия** контекстного меню: Reset Angle, Squeeze, Format, Save PNG — все они используют `selected_text_items()`.

## Fix

**Файл**: `ui/flow_textitem.py`, метод `showFlowContextMenu()` (строка 864)

Добавить `self.setSelected(True)` в начало метода, перед построением контекстного меню:

```python
def showFlowContextMenu(self, scene_pos: QPointF, screen_pos):
    self.setSelected(True)  # <-- ADD THIS LINE
    # ... rest of method unchanged
```

Это единственное изменение — 1 строка.

## Почему это работает

Цепочка вызовов:
1. `mouseReleaseEvent` → `FlowShapeControl.handleContextMenu()` (строка 713)
2. `handleContextMenu` находит FlowTextBlkItem → вызывает `item.showFlowContextMenu()` (строка 770)
3. `showFlowContextMenu` → `scene.build_context_menu()` → показывает меню
4. Пользователь кликает "Reset Angle" → `canvas.reset_angle.emit()` → `onResetAngle()`
5. `onResetAngle()` → `selected_text_items()` → теперь возвращает выделенный элемент

## Затронутые файлы

- `ui/flow_textitem.py` — 1 строка добавляется

## Верификация

1. Открыть комикс, перейти в режим редактирования текста
2. Выбрать FlowTextBlkItem (текстовый блок с flow-точками)
3. Правый клик → "Reset Angle" → угол должен сброситься в 0
4. Undo (Ctrl+Z) → угол должен вернуться
5. Проверить что Delete, Copy, Format, Layout, Translate тоже работают после правого клика
