# Fix: Reset Angle doesn't work on FlowTextBlkItem

## Problem

When right-clicking on a FlowTextBlkItem, FlowShapeControl intercepts the event and shows the context menu. But the TextBlkItem is NOT selected (Qt only auto-selects on left-click). So `selected_text_items()` returns empty, and "Reset Angle" does nothing.

## Fix

**File**: `ui/flow_shapecontrol.py`, `handleContextMenu` method

Select the blk_item when right-clicking on it, so that `selected_text_items()` returns it:

```python
if isinstance(item, FlowTextBlkItem):
    # Select the item so context menu actions (Reset Angle, etc.) work
    if self.scene() is not None:
        self.scene().clearSelection()
    item.setSelected(True)
    item.showFlowContextMenu(scene_pos, screen_pos)
    return True
```
