# Project Brief

## BallonsTranslator — Deep Learning Assisted Manga/Comic Translation Tool

A deep learning-assisted comic/manga translation tool with one-click machine translation and image/text editing capabilities.

### Core Requirements
- One-click machine translation pipeline: text detection → OCR → translation → inpainting → text rendering
- Support Japanese and English comics (manga and Western comics)
- WYSIWYG rich text editing with formatting preservation (color, outline, angle, orientation, alignment)
- Image editing with mask editing and inpainting brush
- Full-text search/replace for original and translated text
- Word document import/export
- Support for vertical (webtoon) comics
- Font style presets

### Key Goals
- Minimize manual effort in comic translation
- Preserve original text layout aesthetics in translations
- Modular architecture for easy extension (detectors, OCR, translators, inpainters)
